# OpenWRT Upgrade for Google Gale Router + Passwall2

با استفاده از این فایل، روتر Google Gale شما به آخرین نسخه **OpenWRT 25.12.5** ارتقا پیدا می‌کند و همچنین پکیج **passwall2** نیز نصب خواهد شد.

---

## 🚀 ویژگی‌ها
- بروزرسانی به OpenWRT 25.12.5
- نصب خودکار Passwall2
- مناسب برای روتر Google Gale
- نصب ساده و سریع

---

## 📥 روش نصب

### 1. انتقال فایل به روتر
ابتدا فایل `sysupgrade` را به مسیر زیر در روتر منتقل کنید:
/tmp/

برای انتقال فایل می‌توانید از ابزارهای زیر استفاده کنید:
- MobaXterm
- FileZilla
- WinSCP

---

### 2. اجرای دستور در روتر
پس از ورود به ترمینال روتر، دستورات زیر را اجرا کنید:

```bash
cd /tmp
sysupgrade openwrt-sysupgrade.bin
⚠️ نکته مهم

اگر روی روتر شما OpenWRT از قبل نصب نیست، باید از فایل با نام factory استفاده کنید:

factory

نه فایل sysupgrade.
📌 توضیحات
فایل sysupgrade برای بروزرسانی سیستم‌های دارای OpenWRT است
فایل factory برای نصب اولیه روی دستگاه‌های بدون OpenWRT استفاده می‌شود
