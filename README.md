# OpenWrt 25.12.5 + Passwall2 برای Google WiFi (Gale)

این ریپو فقط یک **ورک‌فلوی بیلد** است؛ سورس اصلی OpenWrt هر بار از github.com/openwrt/openwrt کلون و کامپایل می‌شود، نه اینکه سورس کامل اینجا نگه‌داری شود.

## مشخصات دستگاه هدف
- مدل: Google WiFi (AC-1304) — نام داخلی OpenWrt: **Gale**
- Target: `ipq40xx`
- Subtarget: `chromium`
- Device id: `google_wifi`
- 512MB RAM / 4GB eMMC

## نحوه استفاده
1. این پوشه را در ریپوی `dreamboxone` روی گیت‌هاب پوش کنید (مسیر `.github/workflows/build.yml` باید همین‌جا بماند).
2. به تب **Actions** بروید → روی ورک‌فلوی `Build OpenWrt + Passwall2 for Google WiFi (Gale)` کلیک کنید → **Run workflow**.
3. مقدار `openwrt_tag` را `v25.12.5` بگذارید (پیش‌فرض همین است).
4. بیلد معمولاً بین ۱ تا ۳ ساعت طول می‌کشد (بسته به لود سرورهای GitHub).
5. بعد از اتمام، خروجی هم به‌صورت **Artifact** روی همان اجرای Action و هم به‌صورت **Release** در ریپو قرار می‌گیرد؛ به دنبال فایل‌های `*factory.bin*` و `*sysupgrade.bin*` بگردید.

## سفارشی‌سازی پکیج‌ها
فایل `config/google_wifi-passwall2.config` را ویرایش کنید تا هسته‌های پراکسی (xray-core، sing-box، ...) را کم/زیاد کنید. هر هسته کمتر یعنی فریمور کوچک‌تر و فضای فلش بیشتر.

## فلش کردن (خلاصه)
چون Google WiFi یک بوت‌لودر مبتنی بر ChromeOS دارد، فلش اولیه با روش‌های معمول OpenWrt فرق دارد:

1. دستگاه باید در **Developer Mode** باشد (طبق مستندات رسمی).
2. برای اولین بار، فایل `*-google_wifi-squashfs-factory.bin` روی یک USB درایو نوشته و از طریق پورت USB-C دستگاه بوت می‌شود.
3. پس از بوت موفق OpenWrt، فریمور روی eMMC داخلی نوشته می‌شود.
4. برای آپدیت‌های بعدی، از فایل `sysupgrade.bin` همان‌طور که در LuCI معمول است استفاده می‌شود.

مراحل دقیق و ابزارهای لازم (کابل USB-C، هاب و ...) در ویکی رسمی OpenWrt توضیح داده شده: https://openwrt.org/toh/google/wifi — پیشنهاد می‌کنم قبل از فلش کردن حتماً همان صفحه را مرحله‌به‌مرحله دنبال کنید، چون یک اشتباه در این مرحله می‌تواند دستگاه را غیرقابل‌بازیابی (brick) کند.

## نکته درباره اپدیت بعدی Passwall2
چون همه‌چیز از سورس بیلد می‌شود، برای گرفتن نسخه جدیدتر Passwall2 کافی است دوباره همین Action را اجرا کنید (فید هر بار آخرین کد را می‌گیرد مگر کامیت خاصی پین شده باشد).
