# OpenWRT for Google Router + Passwall2

با استفاده از این فایل، روتر Google به آخرین نسخه **OpenWRT 25.12.5** ارتقا پیدا می‌کند و همچنین پکیج **passwall2** نیز نصب خواهد شد.

---

## 🚀 ویژگی‌ها
- بروزرسانی به OpenWRT 25.12.5
- نصب خودکار Passwall2
- فقط مناسب برای روتر Google Gale
- نصب ساده و سریع

---

## 📥 روش نصب

### 1. انتقال فایل به روتر
ابتدا فایل `sysupgrade` را به مسیر زیر در روتر منتقل کنید:


/tmp/

برای انتقال فایل به روتر گوگل می‌توانید از ابزارهای زیر استفاده کنید:
- MobaXterm
- FileZilla
- WinSCP

---

### 2. اجرای دستور در روتر
پس از ورود به ترمینال روتر، دستورات زیر را اجرا کنید:


cd /tmp
              
sysupgrade openwrt-sysupgrade.bin


⚠️ نکته مهم

اگر روی روتر شما OpenWRT از قبل نصب نیست، باید از فایل با نام زیر استفاده کنید:


openwrt-factory.bin

نه فایل openwrt-syupgrade.bin

ضمنا اگر روی روتر گوگل، از قبل سیستم عامل OpenWRT نصب نیست برای نصب نیاز به تجهیزاتی دارید و دستوراتی که در آدرس زیر قابل مطالعه است:

https://openwrt.org/toh/google/wifi?s[]=google



📌 توضیح
فایل sysupgrade برای آپدیت سیستم‌های دارای OpenWRT است
فایل factory برای نصب اولیه روی دستگاه‌های بدون OpenWRT استفاده می‌شود
